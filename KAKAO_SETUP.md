# 카카오 공유 기능 설정 가이드

카카오톡 공유 기능을 사용하려면 카카오 개발자 콘솔에서 앱을 등록하고 설정해야 합니다.

## 📋 설정 단계

### 1. 카카오 개발자 계정 생성
1. [Kakao Developers](https://developers.kakao.com/) 접속
2. 카카오 계정으로 로그인

### 2. 애플리케이션 생성
1. 우측 상단 **"내 애플리케이션"** 클릭
2. **"애플리케이션 추가하기"** 클릭
3. 앱 이름 입력 (예: "하늘책방")
4. 사업자명 입력 (개인의 경우 본인 이름)
5. **저장** 클릭

### 3. JavaScript 키 확인
1. 생성된 애플리케이션 선택
2. **"요약 정보"** 탭에서 **JavaScript 키** 복사
3. 프로젝트 루트의 `.env` 파일에 추가:
   ```
   VITE_KAKAO_KEY=여기에_복사한_JavaScript_키_붙여넣기
   ```

### 4. 플랫폼 등록 (중요! ⭐)
1. 좌측 메뉴에서 **"플랫폼"** 선택
2. **"Web 플랫폼 등록"** 클릭
3. **사이트 도메인** 입력:
   - 개발 환경: `http://localhost:5173`
   - 배포 환경: 실제 도메인 (예: `https://yourdomain.com`)
4. **저장** 클릭

> ⚠️ **주의**: 도메인을 등록하지 않으면 "domain" 관련 에러가 발생합니다!

### 5. 활성화 설정
1. 좌측 메뉴에서 **"카카오 로그인"** 선택
2. **활성화 설정** ON (선택사항)

## 🔍 문제 해결

### "domain" 에러가 발생하는 경우
- **원인**: 플랫폼에 도메인이 등록되지 않았습니다
- **해결**: 위의 "4. 플랫폼 등록" 단계를 다시 확인하세요

### "초기화되지 않았습니다" 에러
- **원인**: `.env` 파일의 키가 잘못되었거나, 개발 서버가 재시작되지 않았습니다
- **해결**:
  1. `.env` 파일에 `VITE_KAKAO_KEY` 확인
  2. 개발 서버 재시작 (`Ctrl+C` 후 `npm run dev`)
  3. 브라우저 콘솔에서 "Kakao SDK 초기화 완료: true" 확인

### 이미지가 표시되지 않는 경우
- **원인**: HTTP 이미지는 카카오에서 차단됩니다
- **해결**: 코드에서 자동으로 HTTPS로 변환하도록 이미 처리되어 있습니다

## ✅ 설정 확인

1. 개발 서버 실행: `npm run dev`
2. 브라우저 콘솔(F12) 열기
3. "Kakao SDK 초기화 완료: true" 메시지 확인
4. 책 모달 열고 노란색 카카오톡 버튼 클릭
5. 카카오톡 공유 팝업이 열리면 성공! 🎉

## 📱 배포 시 추가 작업

배포할 때는 추가로 도메인을 등록해야 합니다:

1. 카카오 개발자 콘솔 > 플랫폼
2. **"Web 플랫폼 등록"** 클릭
3. 실제 배포 도메인 추가 (예: `https://yourdomain.com`)
4. `.env` 파일의 키를 배포 환경 변수로 설정
